# MIA-like Portfolio (Next.js)

Desktop-first interactive portfolio prototype inspired by sci-fi map navigation layouts.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## What is included

- Node-map style project selector
- Right-side status and score panel
- Desktop-only guard message for mobile
- Editable project data in `app/page.tsx`
